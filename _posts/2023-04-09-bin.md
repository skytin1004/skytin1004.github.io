## bin 함수 정의
bin함수는 10진수의 정수를 2진수의 문자열로 변환하는 함수입니다.<br>

## bin 함수 용도

bin함수는 10진수를 2진수로 변환할 때 사용합니다.

## bin 함수 사용법

: bin(<span style="color:#9370DB">10진수</span>)<br>


bin함수는 10진수를 정수형이 아닌 문자열로 변환하는 함수이므로 유의해서 사용합니다.<br>

### bin함수 예시<br>

10진수 정수 10을 bin함수에 집어넣고 출력하겠습니다.<br>

bin함수의 결과값이 어떤 타입으로 나오는지 확인하고자 `print(type(result))` 코드를 넣었습니다.

```Python
result = bin(10)

print(result)
print(type(result))

```
실행결과
```Python
0b1010
<class 'str'>

```

bin(10)의 결과값은 "0b1010" 이며 "문자열(str)" 타입 입니다.<br>

2진수 "1010"이라는 값 앞에 붙은 "0b"는 이 숫자가 2진수 임을 알려주는 표시입니다.

## 앞에 붙어있는 '0b' 제거하기

위에서 bin(10)의 결과로 "0b1010"이 나왔습니다. 이때 "1010" 앞에 붙어있는 "0b"를 없애고 싶다면 문자열의 슬라이싱 기능을 이용할 수 있습니다.

```Python
result = bin(10)[2:]

print(result)

```

실행결과
```Python
1010
```

bin함수 뒤에 [2:]를 붙여주면 앞에 있는 "0b"를 제외한 나머지 숫자"1010"이 문자열 타입으로 나옵니다.
