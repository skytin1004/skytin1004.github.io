## sum 함수 정의
sum함수는 리스트 원소들의 총합을 구하는 함수입니다.<br>
<span style="color:gray">※sum 함수는 리스트 뿐만 아니라 딕셔너리, 튜플 등 반복가능한 개체를 인자로 받을 수 있습니다.</span>
## sum 함수 사용법
sum(<span style="color:#9370DB">"리스트"</span>)<br>
<span style="color:gray">※sum 함수에 들어오는 리스트의 원소들은 전부 '숫자'로 이루어져있어야 합니다.</span>
## sum 함수 예시 


#### 예시1: 숫자로 이루어진 리스트
```python
number= [1,2,3,4,5,6]
result = sum(number)
print(result)
```
실행결과
```python
21
```


#### 예시2: 숫자로 이루어진 튜플
```python
number= (1,2,3,4,5,6)
result = sum(number)
print(result)
```
실행결과
```python
21
```
#### 예시3: 숫자로 이루어진 집합
```python
number= {1,2,3,4,5,6}
result = sum(number)
print(result)
```
실행결과
```python
21
```

#### 예시4: 딕셔너리
```python
number= {'a' : 1 ,'b' : 2 ,'c' : 3 ,'d' : 4 ,'e' : 5 , 'f' : 6}
result = sum(number.values())
print(result)
```
실행결과
```python
21
```
