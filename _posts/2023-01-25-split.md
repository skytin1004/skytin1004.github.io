## split함수 정의
split 함수는 문자열을 쪼개서 리스트로 만드는 함수입니다.<br>

## split 함수 사용법
1. <span style="color:#9370DB">"문자열"</span>.split() <span style="color:gray">(띄어쓰기 기준으로 문자열 쪼개기)</span><br>
<span style="color:gray">(ex."Hello my name is skytin1004".split())</span><br>
2. <span style="color:#9370DB">"문자열"</span>.split(<span style="color:#6B8E23">"구분자"</span>)<span style="color:gray">(구분자 기준으로 문자열 쪼개기)</span><br>
<span style="color:gray">(ex."Hello_my_name_is_skytin1004".split("_"))</span><br>
3. <span style="color:#9370DB">"문자열"</span>.split(<span style="color:#6B8E23">"구분자"</span>,<span style="color:#6B8E23">분할횟수</span>)<span style="color:gray">(분활횟수 만큼 구분자 기준으로 문자열 쪼개기)</span><br>
<span style="color:gray">(ex."Hello_my_name_is_skytin1004".split("_",2))</span><br>
4. <span style="color:#9370DB">"문자열"</span>.split(<span style="color:#6B8E23">sep="구분자"</span>,<span style="color:#6B8E23">maxsplit=분할횟수</span>)<span style="color:gray">(3번과 동일)</span><br>
<span style="color:gray">(ex."Hello_my_name_is_skytin1004".split(sep="_",maxsplit=2))</span><br>

## split 함수 예시 

#### 예시1: 문자열.split() 
<span style="color:gray">(띄어쓰기 기준으로 문자열 쪼개기)</span>
```python
myname="Hello my name is skytin1004"
my_list=myname.split()
## split() 괄호안에 입력값이 없으면 띄어쓰기 기준으로 문자열이 나뉩니다.
print(my_list)
```
실행결과
```python
['Hello', 'my', 'name', 'is', 'skytin1004']
```
문자열.split(' ')과 실행결과는 동일합니다.

#### 예시2: 문자열.split('구분자') 
<span style="color:gray">(구분자 기준으로 문자열 쪼개기)</span>
```python
myname="Hello_my_name_is_skytin1004"
my_list=myname.split('_')
## split() 괄호안의 입력값이 '_'이므로 '_' 기준으로 문자열이 나뉩니다.
print(my_list)
```
실행결과
```python
['Hello', 'my', 'name', 'is', 'skytin1004']
```
#### 예시3: 문자열.split('구분자',분할횟수)
<span style="color:gray">(분활횟수 만큼 구분자 기준으로 문자열 쪼개기)</span>
```python
myname="Hello_my_name_is_skytin1004"
my_list=myname.split('_',2)
## split() 괄호안의 입력값이 '_'이므로 '_' 기준으로 문자열이 나뉩니다.
## 분할횟수 2로 지정
print(my_list)
```
실행결과
```python
['Hello', 'my', 'name_is_skytin1004']
```
실행결과를 보면 알 수 있듯이 분할횟수를 2로 지정했을 때 왼쪽부터 두 번 split함수가 적용된 것을 알 수 있습니다.

#### 예시4: 문자열.split(sep='구분자',maxsplit=분할횟수)
<span style="color:gray">(분활횟수 만큼 구분자 기준으로 문자열 쪼개기)</span>
```python
myname="Hello_my_name_is_skytin1004"
my_list=myname.split(sep='_',maxsplit=2)
## split() 괄호안의 입력값이 '_'이므로 '_' 기준으로 문자열이 나뉩니다.
## 분할횟수 2로 지정
print(my_list)
```
실행결과
```python
['Hello', 'my', 'name_is_skytin1004']
```
