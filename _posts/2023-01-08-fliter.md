## filter함수 정의
filter 함수는 입력된 리스트 에서 함수 조건에 맞는 원소들을 추출하는 함수입니다.<br>

<span style="color:gray">※filter 함수는 리스트 뿐만 아니라 딕셔너리, 튜플 등 반복가능한 개체를 인자로 받을 수 있습니다.</span>

## filter 함수 용도
filter함수는 조건에 맞는 데이터를 뽑아낼 때 사용합니다.<br>

## filter 함수 사용법
: filter(<span style="color:#9370DB">함수</span>,	<span style="color:#6B8E23">리스트</span>)<br>
fliter 함수는 <span style="color:#6B8E23">두번째 인자로 들어온 리스트</span> 중에서 <span style="color:#9370DB">첫번째 인자로 들어온 함수</span>에 포함되는 원소들을 추출합니다.<br>

<span style="color:gray">※filter 함수 정리: 첫번째 인자로 들어온 함수가 두번째 인자로 들어온 데이터를 필터링</span>

## filter 함수 예시 
fliter함수를 이용하기 위해서는 <span style="color:#9370DB">첫번째 인자에 들어갈 함수</span>를 정의해야 합니다.<br>
#### 예시1: 0보다 큰 양수 추출
```python
numbers=[-2,0,-5,6,-9,7,3]
def function(n):
	return n>0
result = list(filter(function,numbers))
print(result)
```
실행결과
```python
[6, 7, 3]
```
#### 예시2: 문자열에 들어있는 숫자 추출
```python
myname="Hello my name is skytin1004"
result=list(map(int, filter(lambda u: u.isdigit(), myname))) 
print(result)
##isdigit함수: 문자열이 전부 숫자로만 이루어져있으면 True 리턴
##map함수:두번째 인자로 들어온 myname을 쪼개서 하나하나씩 적용
##filter함수:map에의해 쪼개어진 myname의 각 문자가 숫자인지 판별
```
실행결과
```python
[1, 0, 0, 4]
```
예시2번은 앞으로 서술할 map함수와 isdigit함수에서도 계속 다루겠습니다.
